### 실행방법
powershell 에서 실행함
- julu java sdk 17 설치 필요

```
.\gradlew build; java -jar .\build\libs\untitled-1.0-SNAPSHOT.jar
```

### 가능한 범주
- 폰트 사이즈, 굵게 표현 불가
- 컬러 / 이모지 / 특수문자를 이용한 표 표현 가능
